### 💫 About Me

I blend organic, editorial aesthetics with tactile, interactive interface design. My work bridges the gap between structured user experience research and fluid, playful front-end interactions. 

- 🎨 **Visual Design**: Crafting hand-drawn illustration assets, oil painting, and high-fidelity layouts.
- 💻 **Creative Coding**: Building interactive desktop canvases, spring physics navigation, and 3D web mockups.
- 🌿 **Current Palette**: Inspired by earthy olive greens, terracotta plasters, and minimalist typography.

### 🌟 Featured Project

#### [🖼️ Curated Editorial Portfolio](https://github.com/sandlimaurya/Portfolio)
An immersive, interactive digital workstation portfolio showcasing visual debris, slow-scrolling marquee moodboards, and a curated print shop.

- **Interaction**: Features a draggable desktop window layout, spring-loaded 3D sketchbook simulator, and custom cursor gestures.
- **Commerce**: Built-in Lemon Squeezy, Razorpay, and PayPal checkouts linked to custom mockups.
- **Tech Stack**: Next.js, Framer Motion, Supabase, Vanilla CSS.
