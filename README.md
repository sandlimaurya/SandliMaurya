<h3 align="center">✿ About Me</h3>

<table align="center" border="0" cellpadding="14">
  <tr>
    <td width="55%" valign="top">

I'm a **UX/UI & Visual Designer** who translates ideas into elegant, tangible digital spaces — built on editorial cleanliness, high-contrast layouts, and intentional micro-animation. My background in fine art shapes how I think about composition, negative space, and rhythm long before a single pixel is placed.

I care about the *feeling* of an interface as much as its function — the pause before a hover state resolves, the way a page breathes, the softness of a shadow. Good design, to me, is equal parts research discipline and quiet intuition.

**What I bring to a project**
- A trained eye for typography, color, and editorial hierarchy
- Comfort moving between concept sketches and shipped, coded interfaces
- A process that treats constraints (accessibility, performance, brand) as creative fuel, not friction

  </td>
  <td width="45%" valign="top">

**✿ Design Philosophy**

| | |
|---|---|
| 🌸 **Aesthetic** | organic curves, soft terracotta textures, restrained color |
| 🖋️ **Typography** | editorial pairings, generous whitespace, clear voice |
| 🌿 **Foundations** | oil painting & sketching translated into pixel-precise layout |
| 💻 **Craft** | interactive states, GSAP motion, 3D web structures |
| 🧭 **Approach** | research-led, detail-obsessed, quietly bold |

  </td>
  </tr>
